# СЦЗ — Самоуправляемая Цифровая Зона

Самоуправляемое цензуроустойчивое сообщество «с хребтом»: место с культурой,
которое не держится на одном живом человеке и которое нельзя погасить ни уходом
основателя, ни баном площадки, ни сменой настроения модератора.

> Технология делает место несокрушимым, желанным его делают люди и культура —
> и только в этом порядке.

## Чем это НЕ является

- Не государство и не юрисдикция (код держит эскроу, но не судит намерения).
- Не «второй интернет» и не убежище в стиле Tor.
- Не абсолютный суверенитет — покупаем страховку и плавную деградацию, не неуязвимость.

## Зачем

Страховка от трёх будущих:
1. **Основатель уходит** — место продолжает жить.
2. **Площадку сносят** (бан / ICANN / хостинг по жалобе) — место продолжает жить.
3. **Основатель становится самодуром** — он не может втихую переписать правила под себя.

## Стек (берём готовое, своё не изобретаем)

| Слой | Технология | Роль |
|---|---|---|
| Соцслой / личность | **Nostr** | ключи, профили, чаты, ленты, доски-как-ленты |
| Хранение | **IPFS** (живое) + **Arweave** (архив, потом) | контент жив, пока его пиннят |
| Имена | **свой реестр** | имя → хеш содержимого; нужен судья очерёдности (L2-контракт) |
| Управление / казна | **Gnosis Safe 3/5** + Snapshot | мультисиг руками, голосование надстройкой |
| Доступ | резолвер / нативный браузер (Brave) | имя → CID, без DNS и гейтвеев |

Единственное оправданное «своё» — **реестр имён**: его ценность не в чужой сети,
а в том, кто ручается за имя. Всё остальное — готовые протоколы.

### Первый шаг — не загрузка в IPFS

Первый шаг — сделать место, исчезновение которого было бы потерей.
Инфраструктура несокрушимости защищает пустую комнату ровно до тех пор,
пока в комнате ничего нет.

## Роадмап (кратко)

- **Фаза 0 — Семя:** культура + ядро «конституции» + кто толкует ядро →
  см. [`docs/constitution.md`](docs/constitution.md), [`docs/governance.md`](docs/governance.md)
- **Фаза 1 — PoC** (дни–недели, ноль рублей): имя открывается у чужого человека,
  страница собирается из IPFS не с твоей машины. Честный тест: **выключи свой узел**.
- **Фаза 2 — Инфраструктура без хозяина** (2–4 нед): реестр в L2-контракте,
  защита от сквоттинга, резолвер/Brave, Nostr, мультисиг 3/5.
- **Фаза 3 — Онбординг и экономика** (1–2 мес): кнопка «создать страницу»,
  управление ключами, пиннинг «своих», репутация-ёмкость, голосование по «законам».
- **Фаза 4 — Автономия** (3–6 мес): управление через сообщество, Arweave-архив,
  10+ пиннящих узлов, добровольный выход основателя из мультисига.
- **Фаза 5 — Масштаб:** меш как аварийный режим, подсообщества, свой протокол (только если упёрлись в стену).
