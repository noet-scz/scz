# SCZ — Самоуправляемая Цифровая Зона

SCZ — это цифровая юрисдикция для сообществ: открытые протоколы для идентичности,
коммуникации, хранения и управления, устойчивые к цензуре, захвату и уходу основателей.
Не продукт и не компания, а инфраструктурный слой между интернетом и приложениями.

Полные спецификации: [`docs/`](docs/) (`01_VISION` … `10_ROADMAP`).

> Технология делает место несокрушимым, желанным его делают люди и культура,
> и только в этом порядке.

## Клиент

Главный клиент — **нативное приложение на [Tauri v2](https://v2.tauri.app/)**:
кроссплатформенное (Windows, Linux, Android), малого размера, с Rust-бэкендом для
криптографии и сети и веб-фронтендом для интерфейса. Исходники в [`app/`](app/).

Раньше клиентом было браузерное расширение (заморожено в `poc/zone-portal/` как референс).
Переход на нативное приложение убирает зависимость от магазинов расширений, даёт мобильные
платформы и полноценный доступ к сети. Обоснование: [`docs/09_CLIENTS.md`](docs/09_CLIENTS.md).

## Что умеет сейчас (Фаза 0, «уровень расширения»)

- **Личность** — ключ secp256k1, генерация/импорт/бэкап. Приватный ключ живёт в Rust и
  не покидает устройство.
- **Профиль** — имя, описание, аватар.
- **Лента** — публикация и чтение подписанных сообщений (noet).
- **Сообщества** — создание и участие в пространствах.
- **Вики** — страницы по ключу со связями `[[Имя]]`, версии, удаление.
- **Звонок** — видео P2P (камера и демонстрация экрана), без своих серверов: медиа идёт
  между участниками, сигналинг через публичные реле.

## Архитектура (коротко)

Целевой стек ([`docs/04_ARCHITECTURE.md`](docs/04_ARCHITECTURE.md)): libp2p (DHT, GossipSub),
Hypercore + IPFS, Yjs (CRDT), WebRTC, Tauri. Текущая Фаза-0 реализация прагматична и
совпадает с тем, что уже работало в расширении: сеть и обмен сообщениями через публичные
**Nostr-реле**, медиа через **IPFS-шлюзы**, noet как подписанные события. Миграция на
libp2p/Hypercore/CRDT — по роадмапу ([`docs/10_ROADMAP.md`](docs/10_ROADMAP.md)). Это честная
времянка, не выдаётся за целевой слой.

## Сборка и запуск

Нужны [Rust](https://rustup.rs/) и [Node.js](https://nodejs.org/). На Linux также системные
зависимости WebKitGTK (`webkit2gtk-4.1`).

```bash
cd app
npm install
npm run tauri dev      # запуск в режиме разработки (Linux/Windows desktop)
npm run tauri build    # сборка дистрибутива под текущую ОС
```

Android (нужны Android SDK + NDK + JDK):

```bash
cd app
npm run tauri android init
npm run tauri android build
```

Готовые бинарники под Windows, Linux и Android собираются автоматически в
[GitHub Actions](.github/workflows/) и публикуются в Releases по тегу `v*`.
