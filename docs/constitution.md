# Конституция СЦЗ (ядро) — ЗАГОТОВКА

> **Статус: пусто, ждёт Фазы 0.** Это несменяемое ядро. Меняется только форком,
> не голосованием. Держать узким — всё, что попало сюда, нельзя поправить, не расколов проект.

Двухуровневые правила (см. концепцию §2.2):
- **Конституция** (этот файл) — несменяемое ядро, узкий список. Меняется только форком.
- **Законы** ([`governance.md`](governance.md)) — всё остальное, меняется голосованием.

## Кандидаты в ядро (из концепции, требуют формулировки)

- [ ] **Децентрализация сети** — нет незаменимой машины с рубильником.
- [ ] **Отсутствие единоличного хозяина** — основатель = одна подпись из многих.
- [ ] **Базовые гарантии участника** — (сформулировать какие именно).
- [ ] **Право форка** — несогласные уходят со своей копией всего состояния. Это и есть защита ядра.

## Честный затык (назвать сразу)

Кто-то решает, не нарушает ли конкретная поправка ядро («не централизует ли это сеть?» —
вопрос толкования, а не факт). Тот, кто толкует, **сильнее конституции**. Технически
не закрывается. Закрывается только социально — см. [`governance.md`](governance.md):
кому доверено толкование, как его сменить, и форк как последнее средство.
