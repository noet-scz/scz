noet

Для разработчиков

Собери своё на noet: страницы, форумы, доски, вики. Сервер не нужен.

ГлавнаяЛюди

Конструктор: пять примитивов

На каждой странице noet доступен window.noet — конструктор из пяти примитивов. Из них собирается что угодно, без правок ядра. Данные это подписанные события, живут в сети и переживают любой сервер.

Identity — кто действует

const me = await noet.identity.me();          // { pubkey }
const profile = await noet.identity.get(pubkey);

Space — пространство любого типа

const s = await noet.space.create({ type: "forum", title: "Мой форум" });
const mine = await noet.space.list({ owner: me.pubkey });

Artifact — содержимое

await noet.artifact.publish({ space, content: "первый пост" });
const feed = await noet.artifact.query({ space });

Relation — связи

await noet.relation.add({ type: "reply", to: postId, toAuthor: pk, content: "ответ" });
const replies = await noet.relation.query({ to: postId });

Policy — правила

await noet.policy.set(space, { post: { minRep: 3 } });
const ok = await noet.policy.can({ scope: space, action: "post" });
