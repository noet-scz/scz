# noet — документы экосистемы

noet это не платформа и не «второй интернет». Это **цифровой конструктор**: набор
примитивов, протоколов и инструментов, из которых люди сами собирают форумы, блоги,
доски, вики, маркетплейсы, сообщества — не меняя ядро.

Показатель успеха один: **поверх noet построено больше систем, чем встроено внутрь.**

## Главный вопрос каждого решения

> Я помогаю людям строить своё, или снова делаю платформу вместо конструктора?

Если строю платформу — остановиться и перепроектировать.

## Статус

PoC работает (личность, имена, страницы в IPFS, реле, репутация, делегации доступа).
Эти документы задают переход от «реестр хранит всё» к «реестр это тонкий оракул имён,
всё остальное переносимо». Реализация — по [05-roadmap.md](05-roadmap.md).
