noet

Свободный интернет прямо в браузере.

Свои домены и сайты, привязанные к твоему ключу: их нельзя заблокировать, удалить или отобрать. Центрального сервера нет, каждая страница живёт в сети и подписана тобой. Поставь дополнение один раз.

1 Поставь дополнение

Vivaldi · Chrome · Brave · Edge

Скачать .zip
  1. Распакуй архив в любую папку и не удаляй её.
  2. Открой chrome://extensions (Vivaldi: vivaldi://extensions).
  3. Включи Режим разработчика справа вверху.
  4. Нажми Загрузить распакованное и выбери папку.

Firefox

Скачать .xpi
  1. Открой about:debugging#/runtime/this-firefox.
  2. Нажми Загрузить временное дополнение, выбери .xpi.

Слетает при перезапуске Firefox, пока дополнение не подписано. Для постоянной работы удобнее Chrome или Vivaldi.

2 Создай личность

Нажми иконку noet рядом с адресной строкой и выбери «Создать личность». Дополнение само скачает файл noet-ключ.txt.

Сохрани этот файл. Это твой единственный вход: пароля и восстановления по почте нет. Потеряешь ключ, потеряешь личность и страницы.

3 Пользуйся

Набирай в адресной строке: noet.nt поиск, id.nt твой профиль, people.nt люди, relay.nt лента, dev.nt для тех, кто собирает своё.

Своя страница. На id.nt нажми «Мои домены», займи имя (например alex) и отредактируй страницу. Адрес alex.me станет твоим, можно и поддомены: blog.alex.me.
Только для компьютера: мобильные браузеры пока не дают ставить такие дополнения.
