# PoC — реестр имён + резолвер (§4.3, §4.5)

Открыть страницу **по имени, а не по хешу**, без DNS и без публичных шлюзов:

```
имя  --(реестр-времянка)-->  CID  --(локальный узел IPFS)-->  страница
```

## ⚠️ Честная пометка (из §4.3)

Реестр — **ВРЕМЯНКА**. Судья очерёдности здесь = сам сервер, а значит его хозяин = ты.
Это сознательный компромисс «бесплатно-но-хозяин» для PoC, прямо как предписывает
документ: *«сервер-затычка… помечен „времянка, хозяин это я“».* Спрятанный суверен тут
**назван** — он на этом сервере. Честный вариант без хозяина — смарт-контракт на L2 (Фаза 2).

Автоматизирована **раздача** имён (правило — код). **Разбор споров** («имя моё, а не его»)
не автоматизируется — это людям/мультисигу (`/admin/grant` имитирует решение мультисига).

## Запуск

```fish
# нужен запущенный IPFS-узел (../README.md) на :5001
cd /home/nyx/noet/poc/registry
node registry.mjs &     # реестр  :5050
node resolver.mjs &     # резолвер :8088
```

Открыть в браузере: <http://localhost:8088/> — список имён, клик открывает страницу по имени.

## API реестра

| Метод | Маршрут | Что делает |
|---|---|---|
| GET | `/health` | статус + пометка «времянка» |
| GET | `/names` | все имена |
| GET | `/resolve/:name` | имя → `{cid, owner, ts}` |
| POST | `/register` | занять свободное имя `{name, cid, owner}` |
| POST | `/admin/grant` | выдать резервное имя (имитация мультисига) |

## Правила, зашитые в раздачу (§4.4)

- **Формат:** `label.зона` или `label.zone`.
- **Резерв системных имён:** `gov core admin root noet sys …` + **односложные** (1 буква)
  закрыты от открытой регистрации → только через `/admin/grant` (решение, не «успел»).
- **Очерёдность:** первым пришёл — первым занял; дату/порядок ставит **сервер** (он и судья).

## Чего PoC НЕ покрывает (честно)

- **Браузерный перехват имён.** Здесь резолвер — это HTTP-сервер на `localhost:8088`.
  Чтобы вводить `manifest.zone` прямо в адресной строке (имя вне DNS), нужен либо
  браузерный плагин, либо нативный IPFS-браузер (Brave) — §4.5, отдельный кусок.
- **Без хозяина.** См. пометку «времянка» — честный судья очерёдности это L2-контракт.
